## 你好，世界！

这是我的第一篇博客文章。我搭建这个博客的目的是：

1. **记录日常** — 把生活中值得记住的事情写下来
2. **技术分享** — 整理学到的技术知识，帮助自己和他人
3. **持续输出** — 通过写作倒逼自己学习和思考

## 本站技术栈

本博客使用以下技术构建：

- **Astro** — 高性能静态站点生成器
- **Tailwind CSS** — 实用优先的 CSS 框架
- **Markdown** — 简洁的写作格式
- **Giscus** — 基于 GitHub Discussions 的评论系统
